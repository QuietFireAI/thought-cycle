# Contributing

This is an early (v0.1) project; expect change. Contributions and questions are
welcome.

- Questions, issues, disclosure: **support@quietfireai.com**
- Each skill lives in its own subdirectory with its own tests; run them before
  opening a PR.

Honesty note: this project is about the gap between what a model thinks and what
it says. Contributions that overstate what the tools do — or hide their limits —
are out of scope by definition.
